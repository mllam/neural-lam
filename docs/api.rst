Overview
--------

Neural-LAM provides a comprehensive Python API for weather prediction model development. The main modules are:

- **neural_lam**: Core package initialization and version info
- **neural_lam.models**: Model architectures
- **neural_lam.datastore**: Data handling and storage
- **neural_lam.create_graph**: Graph creation utilities
- **neural_lam.metrics**: Evaluation metrics
- **neural_lam.utils**: Utility functions
- **neural_lam.vis**: Visualization tools

See the sections below for detailed API documentation of each module.
