Welcome to Neural-LAM, a repository of graph-based neural weather prediction models for Limited Area Modeling (LAM).

Overview
--------

This documentation provides a comprehensive guide to using Neural-LAM including:

- Installation instructions
- Quick start examples
- Complete API reference for all modules
- Model architectures and usage
- Data handling and preparation
- Training workflows
- Utility functions

For more information, visit the `GitHub repository <https://github.com/mllam/neural-lam>`_.

Getting Started
---------------

New to Neural-LAM? Start with the :doc:`installation` guide and then explore :doc:`quickstart` for basic examples.

Need help? Check the :doc:`api` documentation for detailed information about available functions and classes.
